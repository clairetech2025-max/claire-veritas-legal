# Veritas Legal

Docker Space package for the Veritas Legal litigation-intelligence mirror.

This Space is intended to run the sanitized public demo and the real FastAPI
application code. It must not include private matters, uploaded evidence,
runtime memory stores, logs, credentials, model weights, or Azure-only paths.

Configure provider endpoints and optional CourtListener credentials through
Hugging Face Space Secrets and Variables.
