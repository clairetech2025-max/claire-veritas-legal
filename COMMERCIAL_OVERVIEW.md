# VERITAS LEGAL Commercial Overview

## What This Is

VERITAS LEGAL is a local-first litigation intelligence workspace built around a grounded evidence workflow:

`Ingest -> Index -> Query -> Trace -> Timeline`

It is not a generic chatbot. It is an operational legal memory and analysis environment for long-running matters with large evidence sets, chronology problems, and provenance requirements.

## Core Capabilities

- Local model execution via `llama.cpp`
- Persistent matter memory
- File, folder, ZIP, OCR, and pasted-evidence ingest
- Grounded search over matter-scoped evidence
- Trace logging and timeline reconstruction
- Packet/draft generation and export
- CourtListener search and ingest bridge
- Creator Mode for house/context questions, separated from legal evidence
- Portable connector payload templates and session resume notes

## Why It Matters

The product solves a real workflow problem for litigators, investigators, and support teams:

- Large evidence sets are hard to orient
- Old matters lose continuity
- Chronology becomes fragmented
- Evidence provenance gets buried
- Chat-style AI tools do not preserve a usable case spine

VERITAS is designed to preserve the case spine while still giving the user AI-assisted search and drafting.

## Differentiators

1. Local-first deployment
2. Matter-scoped memory and retrieval
3. Provenance-preserving ingest
4. Visible workflow state
5. Timeline reconstruction from evidence
6. External legal source bridging
7. Separate creator/house context so non-legal identity does not pollute legal evidence
