# CLAIRE // VERITAS LEGAL

Persistent Litigation Intelligence

## Product Position

This branch is the primary commercial line for offline and private litigation intelligence:

- evidence organization
- timeline reconstruction
- persistent matter memory
- provenance and citation tracing
- contradiction detection
- document intelligence
- grounded answer workspace

## Operational Model

- local-first
- no cloud dependency required
- FastAPI backend
- HTML5 + Tailwind workspace UI
- llama.cpp ready
- future ARE hooks preserved

## Licensing Model

The evaluation flow is intentionally lightweight and professional:

- first-run timestamp
- machine/install fingerprint
- local `license.json`
- 72-hour evaluation window
- activation modal
- read-only fallback after expiration
- future Gumroad / Stripe integration hooks

## Design Rule

Do not split effort across competing products. Use this branch as the consolidated commercial workspace and preserve the legacy ARE and Tkinter systems only as compatibility layers.
